# Volatility

Sends an email, when the given stock price changes dramatically.
## Prerequisites 
* python3
* pip3 

## Manual Steps 
### In user_cred.py file
* Provide [Tradeking secret keys](https://www.ally.com/api/invest/documentation/oauth/).
* Provide Yahoo email-id and password.

## Run Script
```shell
chmod 777 ./setup.sh
./setup.sh
```
